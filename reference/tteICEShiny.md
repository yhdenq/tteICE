# Shiny app for tteICE

This function opens the RShiny app for tteICE. RShiny application can be
used for generating plots and basic analysis results. It provides a
point-and-click interface, so users can obtain results without writing R
code directly.

## Usage

``` r
tteICEShiny()
```

## Value

Rshiny interface

## Examples

``` r
# \donttest{
if(interactive()){
  tteICEShiny()
}
# }
```
